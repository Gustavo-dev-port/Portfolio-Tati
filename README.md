## 💻 Sobre o Projeto
Este projeto é um **Web Site de Portfólio** desenvolvido para uma profissional da área de UI/UX.

O objetivo foi criar uma interface responsiva, rápida e que destacasse os trabalhos visuais da cliente. Utilizei **TypeScript** para garantir a tipagem segura e escalabilidade do código, evitando erros comuns de JavaScript durante o desenvolvimento.
